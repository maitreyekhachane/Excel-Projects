Project Overview:
This project analyzes the performance of Style Advisors across multiple regions using Excel. 
The objective was to evaluate employee performance, identify high-performing regions, track performance trends,
and generate actionable business insights through an interactive dashboard.

Problem Statement:
Bare International conducts regular evaluations of Style Advisors based on various service quality parameters. The goal of this project was to:
1.Classify advisors based on evaluation scores.
2.Compare performance across different regions.
3.Identify regions with the highest proportion of high performers.
4.Monitor performance trends over time.
5.Provide recommendations to improve overall service quality.

KEY INSIGHTS:
1.North Zone shows the highest average evaluation score(~77), indicating strong performance in that region.
2.South zone has the lowest average score(~67), highlighting the need for improvement in that area.
3.North zone has the highest percentage(40%) of High Performers, making it the best-performing region . 
4.Performance trend over time shows a slight decrease, indicating changes in service quality.
5.The majority of employees fall under the Average Performer category, with fewer High Performers, indicating scope for improvement.
